*************************************
ivy: interactive visual phylogenetics
*************************************

``ivy`` is a Python 3 module for the exploration and analysis of
phylogenetic trees and comparative data. Its main feature is providing
interactive tree figures that can be used interchangeably in IPython
consoles and Jupyter notebooks.
