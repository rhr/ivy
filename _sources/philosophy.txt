.. _philosophy:

Philosophy
==========

Make simple things easy and hard things possible.

Make it easy and fun to learn phylogenetics and comparative analysis.

Exploit the strengths of `IPython <http://ipython.scipy.org>`_ and the
rich libraries `scipy <http://scipy.org>`_, `numpy
<http://numpy.scipy.org>`_, and especially `matplotlib
<http://matplotlib.sourceforge.net>`_.

Create a hybrid user interface of magic commands, interactive Python
code, interactive graphics, and a library API for scripting/app
development.

Provide examples and tutorials (e.g., using IPython's `demo` module)
for teaching.

Guiding principles for API design and code organization
=======================================================

* discoverable and intuitive
* self-documenting
* code style emphasizes readability over speed
* keyboard-friendly
